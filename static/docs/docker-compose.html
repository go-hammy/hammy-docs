Running Hammy in Docker Compose

To run Hammy using Docker Compose, follow these steps:

version: '3'
services:
  hammy:
    image: gohammy/hammy
    ports:
      - "9090:9090"
    volumes:
      - ./my-config.conf:/etc/hammy/my-config.conf
